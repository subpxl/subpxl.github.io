Microservices API Gateway Demo

A demonstration of microservices architecture using Traefik as an API gateway with Go, Flask, and FastAPI backends.

Project Structure

├── docker-compose.yml ├── flask-service/ ├── go-service/ ├── fastapi-service/ └── README.md

Quick Start

  1. Clone the repository:
git clone https://github.com/subpxl/apigateway
cd apigateway

Start the services:

docker-compose up -d

Testing the Services

Access the various services at these endpoints:

  • Flask service: http://localhost/flask
  • Go service: http://localhost/go
  • FastAPI service: http://localhost/fastapi
  • Traefik dashboard: http://localhost:8080

Stop the services:

docker-compose down
